Licensing Made Simple
(SaaS)



Lycentia revolutionizes the way software companies issue, monitor, track, and handle licensing for their software applications.

The platform offloads all the license handling implications done by the software vendors to smart autonomous agents running on the blockchain with no downtime, fraud, or any interference.

Ethereum Blockchain



Our platform runs on the decentralized Ethereum network using Smart Contracts.

Ethereum allows censorship resistant execution of smart contracts, which secures users as well as software vendors on our platform and makes our software globally available.
